## 🚀 Deployment via GitHub Pages

### Schritt 1 – Repository erstellen
1. Gehe zu [github.com](https://github.com) und melde dich an
2. Klicke oben rechts auf **„+"** → **„New repository"**
3. Name: `gta6wiki`
4. Visibility: **Public**
5. Klicke **„Create repository"**

### Schritt 2 – Datei hochladen
1. Im leeren Repository auf **„uploading an existing file"** klicken
2. Die Datei `index.html` per Drag & Drop hochladen
3. Commit-Nachricht: `Initial commit: GTA6Wiki`
4. Auf **„Commit changes"** klicken

### Schritt 3 – GitHub Pages aktivieren
1. Im Repository auf **„Settings"** klicken
2. Im linken Menü: **„Pages"**
3. Unter „Branch" → **„main"** auswählen → **„/ (root)"**
4. Auf **„Save"** klicken
5. Nach ~60 Sekunden ist die Seite live! ✅
